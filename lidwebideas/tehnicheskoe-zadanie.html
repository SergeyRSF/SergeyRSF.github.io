Техническое задание

Общие требования

Сайт должен быть разработан с использованием языков HTML,CSS и Javascript. После окончания работ необходимо обеспечить возможность заказчику самостоятельно вносить изменения в структуру и содержимое сайта. По окончании работ Исполнитель обязан предоставить полностью выполненный прототип сайта, дизайн и вёрстку, исходные графические материалы по дизайну и исходные коды по вёрстке.

Предмет разработки

Предметом разработки является Интернет-сайт гильдии «Иззи».
Назначение сайта:
- предоставление информации о гильдии «Иззи»;
- предоставление информации о деятельности гильдии «Иззи»;
- предоставление информации о наборе в гильдию «Иззи»;
- предоставление возможности отправки заявки в гильдию «Иззи»;
Цель создания сайта: привлечение потенциальных участников гильдии.

Требования к дизайну сайта

Основые разделы сайта должны быть доступны с первой страницы. На первой странице не должно быть большого обьёма текстовой информации. Также в дизайне сайта не должно присутствовать много сливающегося текста. Дизайн должен быть выдержан в строгих и мягких тонах. Дизайн сайта должен быть выполнен в рабочей среде, предназначенной для создания дизайна веб-сайта.

Требования к вёрстке сайта

Вёрстка сайта должна быть выполнена с использованием языков HTML и CSS и при необходимости, для создания отдельных элементов страниц, допустимо использование языка Javascript. Сайт должен корректно отображаться в браузерах Microsoft Internet Explorer, Mozilla Firefox, Opera, Safari и пр.
Необходимо создать структуру сайта, состоящую из следующих элементов:

1. "Шапка". В данном блоке необходимо расположить логотип гильдии и название гильдии.

2. Блок отображения главного меню сайта. Данный блок должен содержать перечень всех основных страниц. Меню должно поддерживать иерархическую структуру и должно распологаться по центру сайта.

3. Блок отображения нескольких последних новостей;

4. Блок входа зарегистрированных пользователей на сайт;

5. "Подвал". В данном блоке необходимо разместить краткую контактную информацию.
