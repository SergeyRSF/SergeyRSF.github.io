Техническое задание

Общие требования

От сайта требуется максимальная простота, у пользователя не должно быть вопросов как сделать желаемое действие. Доступ к основной информации должен быть максимально быстрым и внимание пользователя должно концентрироваться на наиболее важных навигационных элементах. Исходя из структуры расположения материалов на сайте, необходимо разработать максимально удобную для данной структуры навигацию. Желательно, чтобы пользователь добирался до нужного ему контента за пару кликов. Для лёгкости восприятия текста должны быть применены правила орфографии и пунктуации. Информация, представленная на сайте, должна соответствовать целевому назначению сайта. Для создания впечатления целостности сайта, страницы сайта необходимо выполнить в едином стиле. Вёрстка сайта должна быть разработан с использованием языков HTML,CSS и Javascript. После окончания работ необходимо обеспечить возможность заказчику самостоятельно вносить изменения в структуру и содержимое сайта. По окончании работ Исполнитель обязан предоставить полностью выполненный прототип сайта, дизайн и вёрстку, исходные графические материалы по дизайну и исходные коды по вёрстке.

Предмет разработки

Предметом разработки является Интернет-сайт гильдии «Иззи».
Назначение сайта:
- предоставление информации о гильдии «Иззи»;
- предоставление информации о деятельности гильдии «Иззи»;
- предоставление информации о наборе в гильдию «Иззи»;
- предоставление возможности отправки заявки в гильдию «Иззи»;
Цель создания сайта: привлечение потенциальных участников гильдии.

Требования к дизайну сайта

Разработка дизайна должна соответствовать типу ресурса. Основые разделы сайта должны быть доступны с первой страницы. На первой странице не должно быть большого обьёма текстовой информации. Также в дизайне сайта не должно присутствовать много сливающегося текста. Изобразительные элементы и даже просто цвет и тон фона не должны мешать читаемости текстовых объектов. Дизайн не должен осложнять работу пользователя, не нагружать его глаз своими графическими элементами и картинками. Дизайн сайта должен быть выполнен в рабочей среде, предназначенной для создания дизайна веб-сайта.

Требования к вёрстке сайта

Вёрстка сайта должна быть выполнена с использованием языков HTML и CSS и при необходимости, для создания отдельных элементов страниц, допустимо использование языка Javascript. За логическое структурирование контента отвечает HTML-разметка, тогда как на CSS возложено всё управление его представлением. Сайт должен корректно отображаться в браузерах Microsoft Internet Explorer, Mozilla Firefox, Opera, Safari и пр.
Необходимо создать структуру сайта, состоящую из следующих элементов:

1. "Шапка". В данном блоке необходимо расположить логотип гильдии и название гильдии;

2. Блок отображения главного меню сайта. Данный блок должен содержать перечень всех основных страниц. Меню должно поддерживать иерархическую структуру и должно распологаться по центру сайта;

3. Блок отображения нескольких последних новостей;

4. Блок входа зарегистрированных пользователей на сайт;

5. "Подвал". В данном блоке необходимо разместить краткую контактную информацию.
